# Mathiko Zoo 🐾

Un jeu éducatif pour apprendre les tables de multiplication, déguisé en gestion d'un refuge d'animaux.

## Lancer en local

```bash
npm install
npm run dev
```

Puis ouvrir http://localhost:5173

## Déploiement

Branché sur Vercel — chaque `git push` sur la branche `main` déclenche un déploiement automatique. URL stable pour les joueurs.

## Mettre à jour le jeu

1. Remplacer le contenu de `src/mathiko.jsx` par la nouvelle version.
2. Commiter le changement (via le site GitHub ou en ligne de commande).
3. Vercel détecte, build, et publie automatiquement en ~30 s.
